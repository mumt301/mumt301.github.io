Using the Last.fm API you will create a dynamic artist page with picture and bio for a given user-entered artist.

Steps

1. Create the basic structure of the page

  • Head and body

2. Create the form to enter the artist name

  • A text field is needed
  • A submit button is also needed

3. Retrieve the artist name data from the form

  • A JS function should be called when the user press the submit button
  • Create a variable with the form's data
  • Retrieve the artist name and store it into a variable

4. Check the Last.fm API call and response to get information about an Artist

  • You may need a Last.fm API key, create one!
  • Create variables for the web service root and the resource access
  • Create the text string with all elements of the query (use .join(""), or '+', or .concat())

5. Use the httpGet() function to create the query and get the resources from Last.fm

  • Try it first in the JS console
  • Use it in your code. Create a variable to store the response of the query
  • Print the query response to the console
  • Examine the response in the console. Assuming the first artist is the one the user is looking for, how to access its MBID?
  • Retrieve the artist's MBID, store it into a variable, and print it to the console

6. Parse the results from Last.fm

7. Display the results
