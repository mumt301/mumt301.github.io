Using the Last.fm API you will create a dynamic artist page with picture and bio for a given user-entered artist.

Steps

1. Create the basic structure of the page

  • Head and body

2. Create the form to enter the artist name

  • A text field is needed (include a label for it)
  • A submit button is also needed

3. Retrieve the artist name data submitted through the form

  • A JS function should be called when the window loads
  • Retrieve all the parameters from the URL
  • Check if the artist name has been submitted (i.e., check if the artist name is within the parameters of the URL)
  • If true, retrieve the artist name and store it in a variable
  • Print the variable in the console to verify that you have retrieved and stored the right information

4. Check the Last.fm API call and response to get information about an Artist

  • You may need a Last.fm API key, create one!
  • Craft the API message: create the text string with all elements of the query (use +, or .join(""), or .concat())
  • Print the query URL in the console to verify that it is a well-formed message

5. Use the httpGet() function from last class to create the query and get the resources from Last.fm

  • The httpGet function takes as parameters the query URL and a callback function that will get the response and process it (in step 6)

6. Parse the results from Last.fm in the callback function

  • Create a variable to store the response of the query
  • Print the query response to the console
  • Examine the response in the console. Assuming the first artist is the one the user is looking for, how to access its MBID?
  • Retrieve the artist's MBID, store it into a variable, and print it to the console. (We will need the MBID later)
  • Retrieve the artist's biography and print it in the console

7. Display the results

  • Append the results to the DOM
