Music APIs

In-class exercise

Create a webpage to return the MusicBrainz ID (MBID) and country for a given user-entered artist. A MBID is a unique identifier for each music entity in the MusicBrainz database.

Steps:

  1. Create form to retrieve user's data
  2. Parse the data submitted through the form to extract the name of the artist
    Remember the documentation about the searchParams property for parsing the parameters of the URL.
  3. Determine the MusicBrainz call you have to make to retrieve artists MBIDs.
    Check https://musicbrainz.org/doc/Development/XML_Web_Service/Version_2
  4. Make the HTTP request call.
    Remember the documentation about the XMLHttpRequest object.
    You can use of the httpGet function used in the solution.
  5. Parse the response from the server
  6. Render the data in the browser
    Need a 'placeholder' element for the results
